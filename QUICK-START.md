## 🎨 Customize in 3 Steps

### Change Wedding Details
1. Right-click on page → "View page source"
2. Find text you want to change
3. Edit in any text editor:
   - Dates
   - Times  
   - Location names
   - Messages
4. Save file
5. Refresh browser

### Change Colors
1. Open `style.css` in text editor
2. Find `:root {` at the top
3. Change color hex codes:
   ```css
   --color-gold: #d4af37;    /* Gold accents */
   --color-brown: #2d1f1a;   /* Dark backgrounds */
   --color-red: #b8453e;     /* Red accents */
   ```
4. Save file
5. Refresh browser

### Change Navigation Text
1. Open `index.html`
2. Find `<nav class="navbar">`
3. Edit menu items:
   ```html
   <li><a href="#home">Home</a></li>
   <li><a href="#details">Details</a></li>
   ```
4. Save and refresh

## 📞 Quick Help

### Website Won't Open
```
1. Make sure index.html is in the folder
2. Double-click index.html (not style.css)
3. Try different browser
4. Check file isn't corrupted
```

### Looks Wrong on Mobile
```
1. Update browser to latest version
2. Clear cache (Ctrl+Shift+Del)
3. Try "portrait" orientation
4. Test in different browser
5. Check DevTools (F12) for errors
```

### Images Not Showing
```
1. Check image file names (exact match)
2. Verify images/ folder exists
3. Images must be JPG or PNG
4. Refresh page (Ctrl+F5)
5. Check browser console (F12) for errors
```

### Something Looks Broken
```
1. Refresh page (F5)
2. Full refresh (Ctrl+F5 or Cmd+Shift+R)
3. Clear browser cache
4. Try incognito/private mode
5. Try different browser
```
