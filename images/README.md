## How to Add Images:

### Step 1: Prepare Your Images
1. Ensure images are in **JPG format** (or PNG/WebP)
2. Resize to recommended dimensions using tools like:
   - Adobe Photoshop
   - Canva (free online)
   - GIMP (free desktop tool)
   - ImageOptim (Mac) or FileOptimizer (Windows)

### Step 2: Add to Images Folder
1. Save images with exact names: `image1.jpg`, `image2.jpg`, etc.
2. Place them in this `images/` folder
3. Ensure file permissions are readable

### Step 3: Verify Loading
1. Open the website in a browser
2. Refresh the page (Ctrl+F5 or Cmd+Shift+R)
3. Check if images appear with proper overlays
4. All sections should display with image backgrounds
