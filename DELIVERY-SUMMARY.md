## 🚀 How to Use

### Immediate Use (No Images)
1. Open `index.html` in browser
2. Website works perfectly
3. Beautiful gradient overlays fill image areas
4. All animations and features functional
5. Share with guests immediately

### With Custom Images
1. Prepare 8 images with exact names
2. Add to `images/` folder
3. Refresh browser (Ctrl+F5)
4. Beautiful image overlays appear
5. Professional appearance

### Customization
1. Edit text in `index.html`
2. Change colors in `style.css` (:root variables)
3. Modify fonts in `index.html` (Google Fonts)
4. Adjust animations speed in `style.css`
5. Add/remove sections as needed

### Sharing
1. Local sharing: Zip folder, send via email
2. Online sharing: Upload to Netlify/GitHub Pages
3. Mobile sharing: Open `index.html` on phone
4. QR code: Generate from online link
